#85

Olá pessoal!

Acredito que essa edição pode ser chamada de "A edição dos financiamentos coletivos" já que trouxe 34 só nessa edição.

E há financiamento para todos os gostos: diferentes idiomas (alemão, inglês, francês, espanhol, português), de Gurus, de sistemas novos, para D&D, para OSR, e por aí vai. Produto novo é o que falta para abastecer sua mesa =)

Claro, trouxe outros links, como a lista de indicações para premiação no Ennie Awards, ideias de aventuras, dicas de como customizar classes bases e muito mais.

Espero que curtam! Até a próxima =)
