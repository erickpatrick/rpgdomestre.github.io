Discussões

Que recursos e magias não funcionam ao batalhar no dito "Teatro da Mente"? (@SlyFlourish, via Twitter - em inglês)

Quais são os pontos mais difíceis de sua usar o ter batalhas apenas via imaginação, sem ajuda de minis, grades de batalha e coisas do tipo? Várias respostas bem bacanas.

Dicas

  • Ligue divindades e patronos ao plano de fundo dos personagens (@SlyFlourish, via Twitter)
  • Coloque duas entras diferentes em sua dungeon para não parecer que são trilhos de trem, com um só caminho (@SlyFlourish, via Twitter)
  • Facções opostas dentro de uma organização inimiga traz grandes possibilidades de interpretação (@SlyFlourish, via Twitter)
  • Anote algumas cenas importantes que deseja ver nas prximas sessões (@SlyFlourish, via Twitter)
  • Construa cenas sem achar que serão combate, exploração ou interpretação. Deixe os jogadores decidirem (@SlyFlourish, via Twitter)
  • Misture monstros e efeitos ambientais apropriados à história (@SlyFlourish, via Twitter)
  • [Se você tem jogadores e está gerando diversão no processo, isso é o que importa] (@SlyFlourish, via Twitter)
  • Seu estilo de narrar deve se adaptar ao número de jogadores (@tionitro, via Twitter)
  • Em viagens de X a Y, adicione 1 ou 2 encontros ou poderá criar uma aventura paralela (@tionitro, via Twitter)
  • Você usar Armadura Pesada? Então pense quando deve tirá-la ou usá-la (@QuestScribe, via Twitter)
  • Você ainda está usando suas roupas ensaguentadas de 2 semanas atrás ou roupas apropriadas para o jantar de gala de hoje a noite? (@SirSolitaire, via Twitter)
  • Crie versões de personagens da Cultura Pop caso tenha dificuldade em criar personagens (@OdsAndBodkins, via Twitter)
  • [Você, mestre, também é um jogador. Não esqueça de se perguntar o que você acha legal numa mesa/campanha] (@EvilSqueegee, via Twitter)
  • Mestres apenas podem focar em um jogador por vez. Você suporta seus amigos jogadores em seus momentos? (@clinton_hillman, via Twitter)
  • Jogadores, vocês também são contadore de histórias (@clinton_hillman, via Twitter)
  • Deixe o silêncio bater e os jogadores entenderem a situação. Pergunte só se o silêncio demorar muito (@kc_halfdust, via Twitter)
  • Tudo que um DM faz para sua campanha é um elo em potencial para planos de fundo dos personagens (@shawnmerwin, via Twitter)
  • Como DM, às vezes se sairá muito bem e outras vez nem tanto. Não se desencorage pelos dias ruins (@richardquiner, via Twitter)
  • Busque toda oportunidade com experiências de outros mestres (@DnDTweets, via Twitter)
  • Drama depende de esperança e sem situações difíceis e incertezas, há menos necessidade de experança (@shawnmerwin, via Twitter)
